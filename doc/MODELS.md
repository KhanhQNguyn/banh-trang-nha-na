# MODELS.md

All Mongoose models in `server/src/modules/`. Every model uses `baseSchemaOptions` which adds `createdAt`, `updatedAt` timestamps and transforms `_id` → `id` on serialization.

---

## Category

**File:** `modules/catalog/models/Category.model.js`

| Field | Type | Required | Notes |
|---|---|---|---|
| `name` | String | Yes | Unique |
| `slug` | String | Yes | Unique; auto-generated from `name` (Vietnamese diacritics stripped) |
| `description` | String | No | |
| `image` | String | No | Image URL |
| `sortOrder` | Number | No | Default `0` |
| `isActive` | Boolean | No | Default `true` |
| `createdAt` | Date | — | Auto |
| `updatedAt` | Date | — | Auto |

## Order

**File:** `modules/order/models/Order.model.js`
    
| Field | Type | Required | Notes |
|---|---|---|---|
| `orderNumber` | String | Yes | Unique; generated by `generateOrderNumber()` |
| `userId` | ObjectId | No | Always `null` (guest checkout only in Phase 1) |
| `customerSnapshot` | Object | Yes | See sub-schema below |
| `items` | [OrderItem] | Yes | See sub-schema below |
| `subtotal` | Number | Yes | Sum of all item subtotals |
| `discountAmount` | Number | No | Default `0` |
| `voucherCode` | String | No | |
| `total` | Number | Yes | `subtotal - discountAmount` |
| `status` | String | Yes | Enum; default `pending_confirmation` |
| `statusHistory` | [StatusHistory] | No | See sub-schema below |
| `note` | String | No | |
| `paymentMethod` | String | Yes | Enum: `cod`, `bank_transfer`; default `cod` |
| `cancelReason` | String | No | |
| `createdAt` | Date | — | Auto |
| `updatedAt` | Date | — | Auto |

**Status enum:** `pending_confirmation` → `confirmed` → `shipping` → `completed` (or `cancelled`)

**customerSnapshot sub-schema**

| Field | Type | Required |
|---|---|---|
| `fullName` | String | Yes |
| `phone` | String | Yes |
| `email` | String | No |
| `address` | String | Yes |

**OrderItem sub-schema**

| Field | Type | Required |
|---|---|---|
| `productSnapshot.productId` | ObjectId | Yes |

| `productSnapshot.image` | String | No |
| `productSnapshot.slug` | String | Yes |
| `variantSnapshot.variantId` | ObjectId | Yes |
| `variantSnapshot.sku` | String | No |
| `variantSnapshot.attributes.size` | String | No |
| `variantSnapshot.attributes.flavor` | String | No |
| `unitPrice` | Number | Yes |
| `quantity` | Number | Yes, min `1` |
| `subtotal` | Number | Yes |

**StatusHistory sub-schema**

| Field | Type | Notes |
|---|---|---|
